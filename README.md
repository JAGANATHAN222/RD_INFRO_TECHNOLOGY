How to Run the Programs
Option 1: Using an IDE
Clone the repository:
git clone https://github.com/<your-username>/RD_INFRO_TECHNOLOGY.git
Open the project in your preferred Java IDE.
Navigate to the desired task folder (for example TASK_1).
Open the main class file (for example Main.java or the file mentioned in that task).
Run the program from the IDE.
Option 2: Using Command Line
Open a terminal / PowerShell in the repository folder.
Go into a task folder, for example:
cd TASK_1
Compile the Java files:
javac *.java
Run the main class (replace Main with your actual main class name):
java Main
Repeat the same steps for other task folders (TASK_2, TASK_3, …, TASK_8).
